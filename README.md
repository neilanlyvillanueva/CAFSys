CAFSys is a web-based anonymous feedback platform for the College of Computer Studies (CCS) of Ateneo de Naga University.

## Overview
CCS students often do not raise concerns formally because there is no official anonymous channel. CAFSys addresses that gap by giving students a secure way to submit anonymous feedback while allowing authorized officers to review, track, and respond to concerns.

## Key Features
- Anonymous feedback submission
- Concern categorization and organization
- Admin review and response panel
- Status tracking for ongoing concerns
- Notification system for responses
